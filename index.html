How to Create and Customize your Own Free eBay Templates in Under Five Minutes!

Writing HTML is pretty boring and you really don't have to

Posted by Juan J. Martinez on August 31, 2017

Have you seen other eBay actions using some fancy html templates and you want to do the same but you do not know how to do it? Well, I have a B.S. in Computer Science and I can tell you something, it is pretty boring and time consuming to write HTML and CSS. That is why I created a pretty useful tool that generates free eBay Templates. If you want to use it right now, you can Login here. If you are clueless on how add html to your eBay listing description, please keep reading.

Generating your first HTML eBay Template

The first thing you have to do in order to create professional ebay listing templates is to create an account at freeebaytemplates.com/app which is 100% free. If you don't really want to create an account, you can login as a guest using the credentials guest@email.com for the email and guest for the password.


Once you are in, you are going to see 12 different theme options for your html ebay template. You can visualize them right from the same page. Once you have found the one you like, just click on select to go to the next step. On step two, you are going to see three different layout options. The first layout is going to have everything in a single column. The second option will generate a template with the product information on the right side and the general listing information on the left. The third option will display everything pretty much the same but in the opposite sides.

Once you have selected the layout that you like the most for your listing template, you are going to be able to see a live preview of your listing template. Every element of the template can be removed in case you do not need it. Once you have your template ready, you will need to click on generate code and this will display the entire HTML and CSS text that corresponds to thea live preview. Now, to make things even simpler, you just have to click on copy to clipboard in order to paste your newly created HTML listing template to your listing.


Adding the HTML Template to your Ebay Listing

The last thing you are missing is just to paste the code generated to your eBay listing description. First, login to your eBay account and open a page to modify or create an eBay listing. Once in that page, go to the part that is named "describe your item". Then, You need to click on the option that says "show HTML editor" and paste the code there. Now, just click on save changes to finally add the HTML eBay template to your listing.

Other Features

This ebay template maker is made with bootstrap. This means that this tool generates responsive html templates. You do not need to worry about the dynamic content policy that eBay has either because this tool generates professional eBay templates that do not include any JavaScript code at all.
